Flash Browser

专为 Apple Silicon 芯片 Mac 打造的轻量级 Flash 内容浏览器，原生运行 SWF 动画与游戏。

项目介绍

Flash Browser 是一款仅支持 Apple Silicon（M1/M2/M3/M4 系列芯片） 的原生 macOS 应用，用于在现代 macOS 上重新运行 Flash 内容。
不依赖老旧浏览器、不依赖 Rosetta 转译，原生高效、干净简洁。

核心特性

• ✅ 原生 Apple Silicon 优化：专为 M 系列芯片编译，性能更流畅

• ✅ 仅支持 Mac：无 Windows、无 x86 版本

• ✅ 直接打开 SWF：支持本地文件打开、拖放加载

• ✅ 播放控制：播放、暂停、缩放、重置

• ✅ 轻量无广告：无捆绑、无数据收集、无后台

• ✅ 现代系统兼容：适配最新版 macOS

系统要求

• 设备：Mac 搭载 Apple Silicon 芯片（M1/M2/M3/M4 及以上）

• 系统：macOS 12.0+

• 不支持：Intel 芯片 Mac、Windows、Linux

快速使用

1. 下载最新 Release

2. 将 Flash Browser.app 拖入「应用程序」

3. 打开后直接拖入 SWF 文件即可运行

技术栈

• 语言：Swift

• 架构：Apple Silicon 原生

• 渲染：基于 Ruffle / Flash 渲染引擎（你可自行替换）
